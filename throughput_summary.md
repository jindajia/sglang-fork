# Throughput Benchmark

> **OTPS**: per-request output tokens/s (`user_tps_mean`)  
> **TTFT**: time to first token in ms (`ttft_mean`)  
> **job_tps**: total output tokens / total elapsed time  
> **per_gpu_tps**: job_tps normalized per GPU
